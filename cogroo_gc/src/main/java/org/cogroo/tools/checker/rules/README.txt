Rules.xsd is the schema that is used to automatically generate the classes
in the model package, executing mvn jaxb:generate

DO NOT EDIT ANY CLASS FROM THE model PACKAGE. SINCE THEY ARE AUTOMATICALLY
GENERATED, YOUR CHANGES WOULD BE LOST IF THE CLASSES ARE GENERATED AGAIN.
